Welcome to OPPPY’s documentation!¶

The Output Parse-Plot Python (OPPPY) library is a python based data analysis library designed to extract, store, and plot information from output and dump files generated by scientific software packages.

Dependencies:
  • Python 3.X
  • argparse
  • pickle
  • numpy
  • scipy
  • matplotlib
  • sphinx (to build documentation)
